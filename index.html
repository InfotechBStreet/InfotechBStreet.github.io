Student project showcase

InfotechChatapp

The good old chatapp, in vanilla javascript? Yes, you heard that right. Students learn the concept of long polling, http requests and manipulating the DOM with javascript.

Analog Clock

Analog clock, built using javascript, css and pure willpower. Not only are students using the Date and setInterval functions, they put their math and logic skills to the test with a simple yet challenging project.

Drumkit

Handling keyboard inputs and playing sounds based on the key pressed. It also has visual effects to show which key was pressed. Students learn to integrate css with javascript

Weather App

Using an API from RapidAPI, students learn to make http requests and manipulate the DOM based on the response. It is a fun project to learn about APIs and how to use them.
